## 2. 카테고리 배치 전략

### 2.1 메인 탭 구성
- 핵심 카테고리 6~7개 노출
- 실시간 트렌드 우선 배치
- 사용자 관심사 기반 순위 조정

### 2.2 서브 메뉴 구성
- 우측 필터 또는 햄버거 메뉴
- 세부 카테고리 제공
- 빠른 접근성 고려

### 2.3 개인화 추천
- 사용자 방문 이력 기반
- 관심사 분석
- 맞춤형 카테고리 추천
