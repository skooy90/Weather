# 메인 페이지 컴포넌트 구현 문서

## 1. 개요
메인 페이지의 주요 컴포넌트 구현 내용을 정리한 문서입니다. 각 컴포넌트의 구조, 기능, 스타일링 등을 상세히 설명합니다.

## 2. 컴포넌트 구조

### 2.1 HeroSection
- 위치: `frontend/src/components/HeroSection/HeroSection.jsx`
- 주요 기능:
  - 메인 페이지 상단 영역 표시
  - 검색 기능 UI 제공
  - 다국어 지원
- 스타일링:
  - styled-components 사용
  - 반응형 디자인 적용
  - 그라데이션 배경
  - 애니메이션 효과

### 2.2 ContentGrid
- 위치: `frontend/src/components/ContentGrid/ContentGrid.jsx`
- 주요 기능:
  - 콘텐츠 그리드 레이아웃
  - 카테고리 탭 기능
  - 다국어 지원
- 스타일링:
  - CSS Grid 사용
  - 반응형 그리드 시스템
  - 탭 전환 애니메이션

### 2.3 ContentCard
- 위치: `frontend/src/components/ContentCard/ContentCard.jsx`
- 주요 기능:
  - 개별 콘텐츠 표시
  - 좋아요/공유 기능 UI
  - 조회수 표시
- 스타일링:
  - 카드 디자인
  - 호버 효과
  - 반응형 이미지 처리

### 2.4 Sidebar
- 위치: `frontend/src/components/Sidebar/Sidebar.jsx`
- 주요 기능:
  - 인기 키워드 표시
  - 파트너 추천
  - 인기 콘텐츠 목록
- 스타일링:
  - 고정 너비 사이드바
  - 섹션별 구분
  - 반응형 디자인
